# 中古マンション 比較リスト

物件を比較検討するためのシンプルなWebアプリです。

## 機能

- 物件情報をカード形式で一覧表示
- ソート機能（番号順 / 月額順 / 神谷町順 / 明治神宮前順 / 面積順）
- 各物件の詳細リンク（SUUMO / cowcamo）

## 表示項目

| 項目 | 説明 |
|------|------|
| 月額計 | ローン返済額＋管理費＋修繕積立金の合計（35年ローン / 変動金利0.375%で計算） |
| アクセス | 神谷町駅・明治神宮前駅への所要時間（路線検索の概算値） |
| チェック項目 | EV有無 / 風呂 / 非タイル（床材） |

## ファイル構成

```
propaty/
├── index.html   # メインHTML
├── style.css    # スタイル
├── app.js       # ソート・表示ロジック
├── data.js      # 物件データ
└── readme.md
```

## 使い方

`index.html` をブラウザで開くだけで動作します。

## 物件データの更新

`data.js` 内の配列を編集してください。
